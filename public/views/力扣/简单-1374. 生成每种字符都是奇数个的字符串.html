简单-1374. 生成每种字符都是奇数个的字符串

linsir 2020-04-20 力扣-简单

# 题目描述

给你一个整数 n，请你返回一个含 n 个字符的字符串，其中每种字符在该字符串中都恰好出现 奇数次 。

返回的字符串必须只含小写英文字母。如果存在多个满足题目要求的字符串，则返回其中任意一个即可。

# 例子

输入：n = 4
输出："pppz"
解释："pppz" 是一个满足题目要求的字符串，因为 'p' 出现 3 次，且 'z' 出现 1 次。当然，还有很多其他字符串也满足题目要求，比如："ohhh" 和 "love"。

# 解题方法

/**
 * @param {number} n
 * @return {string}
 */
var generateTheString = function (n) {
  var str = "";
  var arr = [];
  if (n % 2 == 0) {
    for (var i = 0; i < n - 1; i++) {
      str += "a";
    }
    str += "b";
  } else {
    for (var i = 0; i < n; i++) {
      str += "a";
    }
  }
  return str;
};

# 解题思路

1.判断给出的n是整数还是奇数 2.整数全部返回'a' 3.奇数访问n-1个'a' 和 1 个'b'

var generateTheString = function (n) {
  var str = "";
  var arr = [];
  if (n % 2 == 0) {
    //偶数
    for (var i = 0; i < n - 1; i++) {
      str += "a";
    }
    str += "b";
  } else {
    //奇数
    for (var i = 0; i < n; i++) {
      str += "a";
    }
  }
  return str;
};
