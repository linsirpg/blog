简单-面试题24. 反转链表

linsir 2020-04-15 力扣-简单

# 题目描述

定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。

# 例子

输入: 1->2->3->4->5->NULL 输出: 5->4->3->2->1->NULL

# 解题方法

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function (head) {
  var result = head
  var num = 1
  function change(node) {
    if (node == null) {
      result = null
      return
    } else {
      (change(node.next))
    }
    if (result == null) {
      result = node
    } else {
      node.next = null
      num++
      var currNode = result
      while (currNode.next) {
        currNode = currNode.next
      }
      currNode.next = node
    }
  }
  return result
};
