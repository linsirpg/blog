## 量词-解决重复的读写问题

* 通用模式

量词的通用形式为{m,n}，m,n为数字，限定字符组中字符存在的个数，闭区间，m为下限，n为上限。

| 通用形式 | 描述                           |
| -------- | ------------------------------ |
| {n}      | 匹配字符长度为n                |
| {m,n}    | 匹配字符长度最小为m,最大为n    |
| {m,}     | 匹配字符长度最小为m,最大为无穷 |
| {0,n}    | 匹配字符长度最小为空,最大为n   |


* 常用量词
  
| 常用量词 | 描述                            |
| -------- | ------------------------------- |
| +        | 出现次数大于等于1               |
| *        | 出现次数大于等于0               |
| ?        | 出现次数为0或1                  |
| *?       | 重复任意次，但尽可能少重复      |
| +?       | 重复1次或更多次，但尽可能少重复 |
| ??       | 重复0次或1次，但尽可能少重复    |
| {n,m}?   | 重复n到m次，但尽可能少重复      |
| {n,}?    | 重复n次以上，但尽可能少重复     |
