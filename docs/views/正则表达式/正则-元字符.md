## 元字符

| 修饰符 | 描述                                                   |
| ------ | ------------------------------------------------------ |
| .      | 匹配除了回车符(\r)和换行符(\n)以外的字符               |
| ^      | 元字符匹配一个字符串的起始位置                         |
| $      | 元字符可以匹配字符串的结尾                             |
| \w     | 等价于"[a-zA-Z0-9_]                                    |
| \W     | 它等价于[^a-zA-Z0-9_]                                  |
| \d     | 功能等价于[0-9]]                                       |
| \D     | 功能等价于[^0-9]]                                      |
| \s     | 匹配空格符、制表符、回车符、换行符、垂直换行符和换页符 |
| \S     | 匹配任意不是空白符的字符                               |
| \b     | 匹配字符串的开始或结束                                 |
| \B     | 匹配不是开始和结束的位置                               |
