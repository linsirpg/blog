## 题目描述
请实现一个函数，把字符串 s 中的每个空格替换成"%20"。
## 例子
```
输入：s = "We are happy."
输出："We%20are%20happy."
```

## 解题方法

```
/**
 * @param {string} s
 * @return {string}
 */
var replaceSpace = function (s) {
  return s.split(" ").join("%20")
};
```
## 解题思路
1.使用字符串split方法通过空格切割为一个数组,然后通过数组join方法通过 %20 拼接为一个字符串
