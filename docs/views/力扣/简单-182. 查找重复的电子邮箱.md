## 题目描述
编写一个 SQL 查询，查找 Person 表中所有重复的电子邮箱。

示例：
```
+----+---------+
| Id | Email   |
+----+---------+
| 1  | a@b.com |
| 2  | c@d.com |
| 3  | a@b.com |
+----+---------+
```
根据以上输入，你的查询应返回以下结果：
```
+---------+
| Email   |
+---------+
| a@b.com |
+---------+
```

## 解题方法

```

select Email from  (select Email, count(Email) as EmailCoung from Person group by Email) as static where EmailCoung > 1
```
## 解题思路
子查询 


1.统计所有邮箱的次数
2.统计邮箱次数大于1的邮箱
