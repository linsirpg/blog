## 题目描述
给你一个整数 n，请你返回 任意 一个由 n 个 各不相同 的整数组成的数组，并且这 n 个数相加和为 0 。
 
## 例子
```
输入：n = 5
输出：[-7,-1,1,3,4]
解释：这些数组也是正确的 [-5,-1,1,2,3]，[-3,-1,2,-2,4]。

```
```
输入：n = 3
输出：[-1,0,1]

```


##  解题方法

```
/**
 * @param {number} n
 * @return {number[]}
 */
var sumZero = function (n) {
  var arr = []
  if (n % 2) {
    arr.push(0)
  }
  for (var i = 1; i <= Math.floor(n / 2); i++) {
    arr.push(i)
    arr.push(-i)
  }
  return arr
};
```
##  解题思路

1.判断是否奇数,是的话先往返回的数组里加零

```
var arr = []
if (n % 2) {
  arr.push(0)
}
```
2.因为奇数已经补充0剩下的偶数个直接一半-1一半1
```
for (var i = 1; i <= Math.floor(n / 2); i++) {
  arr.push(i)
  arr.push(-i)
}
```
